📋 Task Endpoints

GET /api/tasks

Get all tasks (supports filtering: ?category_id=xxx&priority=xxx&search=xxx)

GET /api/tasks/:id

Get a specific task by ID

POST /api/tasks

Create a new task

PUT /api/tasks/:id

Update a task

PATCH /api/tasks/:id/toggle

Toggle task completion status

PATCH /api/tasks/:id/timer

Update task timer

DELETE /api/tasks/:id

Delete a task

🏷️ Category Endpoints

GET /api/categories

Get all categories with task counts

GET /api/categories/:id

Get a specific category

POST /api/categories

Create a new category

PUT /api/categories/:id

Update a category

DELETE /api/categories/:id

Delete a category

📊 Statistics Endpoints

GET /api/stats

Get task statistics and analytics

GET /api/stats/export/json

Export all tasks and categories as JSON
